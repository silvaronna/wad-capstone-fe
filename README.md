
# 🌌 Cakrawala Task Management Frontend

### Azka Abdillah (24120400001)

This is the frontend component of the Cakrawala Task Management System, built with **React** and **Vite**.

## 🚦 Setup and Execution

To run this frontend project from scratch, please refer to the main setup guide in the root directory:
👉 **[Fullstack README.md](../README.md)**

### Quick Start:
1. Make sure the backend server is running on `http://localhost:3000`.
2. Install package dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Access the web dashboard at `http://localhost:5173`.
